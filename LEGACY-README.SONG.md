─────────────────────────────────────
(一) 如何設定點歌系統
─────────────────────────────────────

    建立兩個新看板：
        OrderSongs   ※ 點歌紀錄版
            建議閱讀權限：--------------------------------
            建議發表權限：-------------------------------S
        Request      ※ 點歌系統投稿區
            建議閱讀權限：--------------------------------
            建議發表權限：----j---------------------------
    並在 Request 的精華區中，新增 SongBook 的 Class ，然後將歌本依照您的
    規劃擺放進去即可。

         一旦經採用，除了可以讓大家使用您的作品外，
         您的點歌次數也會增加二次。

 一. 文章的標題請明確標示演唱者的姓名以及歌名。若是台語歌、粵語歌，
     也請標明。
     【範例一】[投稿]梁詠琪     短髮
     【範例二】[投稿]高明駿＋王馨平 今生註定
     【範例三】[投稿]江  蕙     感情放一邊(台語)
     【範例四】[投稿]許茹芸     哭牆(粵語)

 二. 歌詞稿件的內容請控制在二十行之內(含<~Src~>點給<~Des~>：<~Say~>那行)。

 三. 稿件的內文也請務必標明歌手、歌名。

 四. 請務必在內文中加上<~Src~>點給<~Des~>：<~Say~>。特別注意每個符號，
     不要打錯。可做適度變化，如加上顏色控制碼、改變順序。
    【範例一】<~Src~> 點給 <~Des~>：<~Say~>
    【範例二】<~Src~> 想對 <~Des~>說：<~Say~>
    【範例三】<~Des~>，<~Src~>想對你說：<~Say~>

     其中<~Src~>這個控制碼在點歌時將會出現點歌者的ID，
     <~Des~>這個控制碼在點歌時將會出現所點給的人，
     而<~Say~>這個欄位在點歌時將會出現所要說的話。
     要將這三個控制碼在內文中如何排列請自行運用。

 五. 請在內文結束後的下一行打上<~End~>，
     <~End~>此控制碼在點歌時會自動將該行後面所有內容刪除。

 六. 請於稿件內文後<~End~>的下一行，標明作者、發表時間。
    【範例一】作者：honey
              時間：Sun Apr 16 2000
    【範例二】作者：lover
              時間：4.16.2000

 六. 稿件請一律上色，黑白稿件恕不受理。關於上色技巧，可參閱asciiart版精華區。

 七. 歌本之內已有之歌曲，仍可再投稿。但投稿前請先到精華區看歌本之中已收錄
     哪些歌曲，盡量投稿歌本裡面沒有的。
