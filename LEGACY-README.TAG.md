Tag Guide                                                      [WindTopBBS]
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

                          WindTopBBS Ver 3.02

                             標籤Tag說明

               Modified by 元智資工  沈俊興(statue.bbs@bbs.yzu.edu.tw)

=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

        Tag 中文翻成「標籤」好了，被打上標籤的文章或信件，
        可以採用「整批作業」的方式，相當方便。

        Tag (標籤) 的相關功能如下：
  ┌──────────────────────────────────┐
  │                                                                    │
  │ ■ 「普通級」的一般指令：                                          │
  │                                                                    │
  │    [t]  tag/unatg  切換選擇要不要標記游標所在文章，                │
  │                    被打上標籤的文章，前面會顯示 '*' 的符號         │
  │                                                                    │
  │    [^T] thread     將同主題之文章，通通打上標籤                    │
  │                                                                    │
  │    [^C] clear tags 清除所有的標籤                                  │
  │                                                                    │
  │    [F]  forward    可以將有標籤的文章，整個打包起來，一起寄回家。  │
  │                    操作的時候，螢幕上最下面一行會顯示：            │
  │                    (A)rticle (T)ag (Q)uit                          │
  │                                                                    │
  │                    A ==> 只寄回游標所在之文章                      │
  │                    T ==> 寄回整個 tag list                         │
  │                    Q ==> 取消                                      │
  │                                                                    │
  │                    並且，你可以選擇需不需要將文章 uuencode、       │
  │                    需不需要更改 E-mail address。                   │
  │                                                                    │
  │ ■ 底下是「輔導級」的指令，限於「板主」：                          │
  │                                                                    │
  │    [c]  cite       可以將有標籤的文章，一起放進「精華區」          │
  │                                                                    │
  │    [^D] prune      將標籤起來的所有文章，通通刪除，                │
  │                    還會送出 cancel message，達成「連線砍信」       │
  │                                                                    │
  │ ■ 底下是「限制級」的指令，限於「站長」：                          │
  │                                                                    │
  │    [^X] terminator 灌水終結者，與目前主題同標題的文章，跨板砍除，  │
  │                    清得一乾二淨，也會送出 cancel message，連線砍信 │
  │                                                                    │
  └──────────────────────────────────┘

     □ 上述功能，在閱讀「個人信件」的時候，同時有效，
        此外，在個人信箱中，一般使用者，即使不是板主，
        也可以按 [^D] 以整批作業的方式，清理自己的信箱。

     □ 目前的設計是，一次最多可以打上 128 個「標籤」，
        當你的標籤超過 128 個，就會「嗶嗶」一聲通知你，
        風之塔已改為 256 個。
